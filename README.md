# ☄️ HELLFIRE VAULT 🎲

## 📜 The Pitch
Welcome to the **Hellfire Vault** — a centralized arcade terminal built for Hawkins High.

This application integrates three systems:

### 1. Ledger Resolver
Upload a debt CSV → calculates minimum transactions.

### 2. Briefing Generator
Generates `.pptx` slides automatically.

### 3. File Sharing (LAN)
Download files over same Wi-Fi.

🧠 AI Feature: Gemini AI answers queries.

---

## 🛠️ Tech Stack
- Python (Flask)
- HTML, CSS, JS
- python-pptx
- Gemini AI
- Supabase (optional)

---

## 🕹️ How to Run

### Clone
```bash
git clone https://github.com/bindusmita49/HELL_FIRE_VAULT.git
cd HELL_FIRE_VAULT
```

### Install
```bash
pip install flask python-pptx python-dotenv google-generativeai supabase
```

### Run
```bash
python app.py
```

Open this folder in your local host
